# Persona Addition Checklist (OctoAcme)

Use this checklist when proposing a new persona or updating an existing persona entry in docs/octoacme-roles-and-personas.md.

- [ ] Role name (clear and concise)
- [ ] Role summary (1–2 sentences)
- [ ] Responsibilities (3–6 bullet points, actionable)
- [ ] Goals / success criteria (how we measure success for this role)
- [ ] Typical communication & interactions (who they work with and how)
- [ ] Examples of responsibilities or deliverables (optional)
- [ ] Conflicts/overlaps with existing roles (note any handoff points)
- [ ] Suggested owner for onboarding and handover (person/team)
- [ ] Acceptance criteria: content aligns with existing docs and improves clarity
- [ ] Stakeholder review completed (if required)

When adding content, keep entries short and focused on ownership, interactions, and measurable outcomes.
